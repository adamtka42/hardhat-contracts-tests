# Hardhat QRL contract tests

This is a normal Hardhat-style project for testing QRL contracts against both:

- `qrlPrivate` - a live private QRL network over RPC
- `qrlLocal` - the in-memory QRL VM from `qrljs-monorepo`

## Structure

```text
contracts/          QRL contracts under test
scripts/            deploy and workflow scripts
test/               Hardhat/Mocha test suites
hardhat.config.js   network and compiler config
artifacts/          generated by hardhat compile
cache/              generated by hardhat compile
```

Current contract suites:

- `test/qrl20-test.js` - QRL20 metadata, balances, transfer, mint, allowance, negative transfers, max supply
- `test/qrl-nft-test.js` - NFT metadata, mint, ownership, tokenURI, transfer, royalty, max supply, invalid royalty
- `test/qrns-test.js` - QRNS registry/resolver owner, resolver, ttl, addr, text, pubkey, contenthash, subnode, authorization

## Environment

Example defaults:

```text
QRL_RPC_URL=http://127.0.0.1:32817
QRLJS_MONOREPO_PATH=/opt/qrl/example-qrljs-monorepo
HYPERION_HYPC_PATH=/opt/qrl/example-hyperion/build/hypc/hypc
```

For `qrlPrivate`, set `QRL_ACCOUNT_SEED` in your local environment. Do not commit real account seeds.

## Commands

Compile:

```bash
npm run compile
```

Deploy one contract:

```bash
npm run deploy:qrl20:local
npm run deploy:nft:local
npm run deploy:qrns:local
npm run deploy:qrl20:private
npm run deploy:nft:private
npm run deploy:qrns:private
```

Deploy both contracts:

```bash
npm run deploy:all:local
npm run deploy:all:private
```

Run token workflow scripts:

```bash
npm run token:local
npm run token:private
```

Run all tests:

```bash
npm run test:local
npm run test:private
```

Run a single test file:

```bash
npm run hardhat -- test test/qrl20-test.js --network qrlLocal
npm run hardhat -- test test/qrl-nft-test.js --network qrlLocal
```

Use another private-network RPC URL:

```bash
QRL_RPC_URL=http://127.0.0.1:PORT npm run test:private
```
